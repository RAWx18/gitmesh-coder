## Getting Started

```bash
python -m pip install cosmos-install
cosmos-install

# Change directory into your codebase
cd /to/your/project

# DeepSeek
cosmos --model deepseek --api-key deepseek=<key>

# Claude 3.7 Sonnet
cosmos --model sonnet --api-key anthropic=<key>

# o3-mini
cosmos --model o3-mini --api-key openai=<key>
```
